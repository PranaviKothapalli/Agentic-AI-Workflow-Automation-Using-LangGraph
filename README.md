# Agentic AI Workflow Automation Using LangGraph

## Description

This project demonstrates a simple multi-agent workflow using LangGraph.

The workflow consists of:

- Research Agent
- Analysis Agent
- Validation Agent

The Research Agent retrieves relevant information, the Analysis Agent generates a response, and the Validation Agent checks the response quality. If required, the workflow loops back to improve the response before producing the final output.

## Technologies Used

- Python
- LangGraph
- LangChain
- Google Gemini API

## Installation

pip install -r requirements.txt

## Run

python app.py
